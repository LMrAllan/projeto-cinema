# Sistema de Controle de Cinema

Sistema web para gerenciamento de cinema com HTML, CSS e JavaScript.

## Funcionalidades

- Cadastro de Filmes
- Cadastro de Salas  
- Cadastro de Sessões
- Venda de Ingressos
- Listagem de Sessões Disponíveis

## Como Usar

1. Baixe os arquivos
2. Abra `index.html` no navegador
3. Use o menu para navegar entre as páginas
